# Portfolio

A modern portfolio website showcasing my skills and projects as a full-stack developer.

## About

This portfolio is built with React, TypeScript, and Vite to demonstrate my expertise in modern web development. It features a clean, responsive design that highlights my technical abilities and professional work.

## Features

- **Projects Showcase** - A curated collection of my development work with live demos and source code links
- **Skills & Technologies** - Overview of my technical stack and competencies
- **About Me** - Introduction to my background and experience as a full-stack developer
- **Responsive Design** - Optimized for all devices and screen sizes

## Tech Stack

- React 18
- TypeScript
- Vite
- CSS3
